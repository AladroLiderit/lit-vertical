Odoo 8 Point of Sale Extension

Assign unique serial number from Point of Sale Frontend


Note: This module is tested and working in community edition.

How to use

  • First of all, we have to enable serial number feature in Settings > Warehouse. Then as per below image, we can set Full Lots Traceability in product form so that during sale from Point of Sale it will ask to enter serial number forcefully.


  • Now we have created Purchase Order of 4 quantity of above product and going to receive them from Incoming Shipment.


  • When we make transfer from incoming shipment, system will automatically split all products into separate lines and user can enter unique serial number as per his requirement.


  • Here is the list of serial number we have created.


  • Next step is to add products into cart from POS. User will get popup automatically for related serial numbers of this product with available quantity.


  • User is not allow to enter the same product more than one time if already added in the list.


  • The same serial number will be displayed on pos receipt also and same will be transfered to the backend pos order, and same time delivery order will be done.






  • Next time if we try to add same product from pos then we can see one quantity is already decreased from hat serial number.
